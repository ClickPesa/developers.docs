---
title: "Payout API Overview"
description: "Understand ClickPesa Payout API and how to integrate in your application"
---

The Payout API allows you to create a full custom experience on collecting payouts with ClickPesa from different payout methods.

- No redirecting customers from your website or application
- Designed for a seamless payout experience
- Ideal for platforms that require a fully customized payout solution with granular control over transactions
- Supports multiple payout methods including mobile money and card payouts
